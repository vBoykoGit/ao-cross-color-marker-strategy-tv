# AO Single Bar Strategy

TradingView стратегия на Pine Script v6, построенная на основе сигнала **Single Bar Pattern** из индикатора [ao-cross-color-marker-tv](https://github.com/ao-cross-color-marker-tv).

---

## Логика стратегии

### 1. Сигнал входа — Single Bar Pattern

Паттерн ищется на гистограмме Awesome Oscillator (AO):

- Берутся три последних закрытых бара AO: `colorPrevPrev`, `colorPrev` (одиночный), `colorNow`
- Паттерн срабатывает, когда средний бар (`colorPrev`) имеет цвет, отличный от соседей, а крайние бары одинаковы

```
Пример (bearish): green → red → green  ← паттерн при ao[2] > 0
Пример (bullish): red → green → red    ← паттерн при ao[2] < 0
```

> **Важно:** цвет одиночного столбца не определяет направление сделки.

### 2. Фильтр силы сигнала («выше/ниже среднего»)

Сигнал принимается только если AO-столбец сигнального бара (`ao[2]`) достиг или превысил среднее значение своей зоны (рассчитывается за `aoAvgLen` последних баров):

| Условие | Сигнал |
|---|---|
| `ao[2] > 0` И `ao[2] >= avgPos` | **strongSell** → SHORT |
| `ao[2] < 0` И `\|ao[2]\| >= \|avgNeg\|` | **strongBuy** → LONG |

### 3. Размещение ордера — SELL/BUY STOP с трейлингом

При получении сигнала размещается стоп-ордер:

- **strongSell** → SELL STOP на `low` сигнального бара, стоп-лосс на `high` того же бара
- **strongBuy** → BUY STOP на `high` сигнального бара, стоп-лосс на `low` того же бара

Пока ордер не исполнен, каждый бар ордер обновляется:

- SELL STOP → `low[1]` (минимум предыдущего бара), стоп-лосс → `high[1]`
- BUY STOP → `high[1]` (максимум предыдущего бара), стоп-лосс → `low[1]`

Это позволяет следовать за ценой: если цена идёт против сигнала, ордер трейлится к каждому новому бару.

### 4. Выход из позиции

| Условие | Действие |
|---|---|
| Цена пересекает MA34 (SMA 34) | Закрыть по рынку (TP) |
| Стоп-лосс выбит | Закрыть по стоп-цене |

Противоположный сигнал при наличии отложенного ордера — отменяет текущий ордер и открывает новый в обратном направлении.

## Параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| `AO Average Lookback` | 100 | Период расчёта среднего AO для фильтра силы |
| `MA Exit Length` | 34 | Период скользящей средней для выхода (синяя линия) |
| `commission_value` | 0.1% | Комиссия брокера |
| `slippage` | 1 тик | Проскальзывание |

---

## Установка в TradingView

1. Открыть TradingView → **Pine Script Editor** (кнопка внизу экрана)
2. Вставить содержимое файла `AO_SingleBar_Strategy.pine`
3. Нажать **Add to chart**
4. Стратегия будет добавлена как overlay (на ценовом графике)

### Рекомендуется подключить совместно с индикатором

Добавить [AO Cross & Color Markers](../ao-cross-color-marker-tv/AO_Cross_Color_Markers.pine) в отдельной панели — для визуального подтверждения сигналов на гистограмме AO.

## Визуализация

- **Синяя линия** — MA(34), линия выхода из позиции
- **Красный треугольник вниз** (над баром) — сигнал strongSell
- **Зелёный треугольник вверх** (под баром) — сигнал strongBuy
- **Красные/зелёные горизонтальные линии** — уровень отложенного ордера и стоп-лосса (пока ордер не исполнен)
